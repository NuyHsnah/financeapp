# Finance App - Tugas E-Learning

Berikut adalah aplikasi **Finance App** yang dibuat sebagai tugas mata kuliah **Pemrograman Mobile / Flutter**. Aplikasi menampilkan **splash screen** dan **menu utama** dengan fitur saldo, kartu ATM, menu cepat, pencarian transaksi, dan daftar transaksi terakhir.
